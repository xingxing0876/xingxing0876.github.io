## 项目背景

希望拥有一个轻量、可版本管理的个人站点，无需服务器即可部署。

## 技术亮点

- 使用 Astro Content Collections 管理项目与文章
- GitHub Actions 实现 push 后自动部署
- 响应式布局，移动端友好
