## 项目背景

面向港美股市场的投资管理平台，由实时行情、策略交易、账目分析、机器手四大核心模块组成，支持 80+ 量化因子与多种交易策略。

## 技术亮点

- 基于 Netty 接收实时行情，Disruptor 无锁队列实现毫秒级端到端延迟
- 搭建量化因子引擎，支持技术指标与自定义因子热加载
- 设计统一券商 API 抽象层，实现订单路由、持仓同步与对账

## 个人职责

技术负责人 / 核心架构设计，负责后端研发团队管理与技术方案评审。
