## 项目背景

由因子中心、模型训练、AI 交易大脑三大模块组成，通过因子回测与模型训练挖掘高价值因子，实时预测股票走势并自动完成交易决策。

## 技术亮点

- 基于 LangGraph 构建 Agent 循环引擎，集成 Function Call、记忆模块与技能库
- 处理港股十年历史行情数据，完成数据清洗与特征工程
- 设计 Agent 与交易系统的安全隔离层，可迁移至智能投顾场景

## 个人职责

技术负责人，负责 Agent 系统整体技术架构与因子中心搭建。
